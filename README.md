# tju_MasterDr_Latex_template

 Tianjin University thesis latex template for personal use. Based on 2021 revised version.



编译环境：xelatex + bibtex
将此设置添加到Latex Workshop的recipes设置中


```
"latex-workshop.latex.recipes":
[
    {
        "name": "xelatex ➞ bibtex",
        "tools": [
            "xelatexmk",
            "bibtex"
        ]
    }
]
```
